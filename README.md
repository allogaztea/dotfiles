# Dotfiles
###### par gaztea

A set of dotfiles to debian based distros and fedora. 

## Common Packages
A couple of packages are common to both distro dotfiles, are:

**Shell**
- Fish

**Code Editors**
- NeoVim
- VSCodium

**Browsers**
- Lynx
- Chrome
- Firefox

**Communication**
- Mull

**Other**
- NodeJS
- Docker
- Neofetch
- Nerd Fonts

### Debian Exclusive
- Signal
