# Dotfiles
###### par gaztea

A set of dotfiles to debian based distros and fedora. 

## Common Packages
A couple of packages are common to both distro dotfiles, are:

**Code Editors**
- NeoVim
- VSCodium

**Browsers**
- Lynx
- Chrome

**Communication**
- Mull

**Others**
- Yarn
- Xclip
- Docker
- NodeJS
- Neofetch
- Nerd Fonts

### Debian Exclusive
- Signal
- Zshell + Plugins

### Fedora Exclusive
- Fish
